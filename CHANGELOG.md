# 版本记录

本项目采用语义化版本号。所有版本均为公开机理约束下的代理仿真软件版本，不代表真实型号数字孪生或官方实现。

## v0.5.0 — 电影级三维表现与自适应画质（2026-07-19）

### 主要修改

- 三维视口拆分为火箭、回收船、海洋天气、连续四索、粒子和电影镜头组件，并使用稳定的 ACES 色调映射与原生抗锯齿路径。
- 新增随 GitHub Pages 离线发布的风暴后暖光天空、海雾浪花图集和箭体污损遮罩；运行时不访问 CDN。
- 重建分段箭体、上部挂索加强环、发动机簇、开放式回收井、钢桁架、绞盘、导向轮、护栏和湿甲板材质。
- 海面由回放时间和场景 seed 驱动，四索在接触前保持统一平面，接触后按载荷趋势表现受力弯曲与回弹。
- 增加自动/高/中/低三维画质偏好、30 秒迟滞、自适应性能监测、降级路径和减弱动态支持。
- Windows ANGLE/D3D 实测将高档实际渲染目标限制为 1.25 DPR、1024 阴影与 128² 海面网格；更高组合会产生黑白帧，因此未作为可用能力宣称。
- 显示层版本升级到 v0.5.0；仿真模型、场景 schema、控制算法和配置指纹保持 v0.4.0/v3 语义不变。

### 边界

所有视觉结构、材质、海况和变形均为解释性代理，不代表真实型号结构或实测摄影数据；视觉挂索点仍不等同于质心等效接触点。

## v0.4.0 — 协同控制、约束 MPC 与算法工作台（2026-07-18）

### 主要修改

- 交会预测改为制导感知滚动预测，网中心控制使用带限 PD 与可达性检查。
- PREPARE/READY/COMMIT 绑定窗口和修订版本，协调器必须收齐箭体与四绞盘同版本确认。
- 接触后四绳局部 PI 张力环驱动受限主动阻尼，记录目标、实际、误差和饱和状态。
- 增加确定性分层约束 MPC、热启动、约束激活诊断和预测 PD 安全回退。
- 新增算法主工作台：显示闭环数据流、3σ 可达域、网中心响应、四绳张力、READY 门控和 MPC 求解健康度。
- 场景与交换文档升级到 schema/document v3，继续严格迁移旧格式并拒绝未知字段。

### 边界

所有预测器、控制器、张力和 MPC 参数均为研究假设或代理标定，不代表长征十号乙真实参数或官方算法。

## v0.3.1 — 可信度修复与 GitHub Pages 发布（2026-07-14）

### 中文版本描述

本版本集中修复 v0.3.0 审查中发现的场景完整性、数值验证、敏感性解释和前端状态一致性问题，并建立从 `main` 自动构建、测试和发布 GitHub Pages 的网页交付链路。

### 主要修改

- 当前 v2 场景改为严格完整校验：必填字段缺失、未知字段、非法采样率和配置指纹不一致均在导入阶段拒绝；仅旧 v1 场景允许按显式迁移规则补默认值。
- 步长收敛实验关闭随机阵风、传感器噪声和随机链路事件，并在共同物理结束时刻比较 Δt、Δt/2、Δt/4，避免把不同随机轨迹误当作积分误差。
- 接触阶段能量账本移入物理 tick，拆分平动、转动、重力—势能及接触储能—阻尼闭合残差。
- 局部敏感性改为双侧扰动；零值参数使用明确绝对扰动下限；终态模式切换单独报告，不再伪造连续敏感性系数。
- 三维场景和运行结果始终使用 `run.config`，编辑中的 `draftConfig` 单独显示，消除旧结果与新参数混用。
- 追溯表计算前后保持同一参数集合，并新增模式边界提示。
- 新增 GitHub Pages 自动部署工作流，发布前执行依赖安装、类型检查、全部测试和生产构建。

### 验证

- TypeScript 类型检查通过。
- 10 个测试文件、67 项测试通过。
- 普通生产构建与 GitHub Pages 子路径构建通过。
- 标称步长收敛、物理 tick 能量账本和双侧敏感性命令行烟雾测试通过。

## v0.2.0 — 接管基线与完整实验闭环（2026-07-14）

### 版本描述

将原有仿真内核、三维场景和遥测组件连接为可运行、可解释、可重复实验的完整前端工作流，作为后续故障实验、验证工具和参数追溯能力的稳定基线。

### 主要修改

- 接通场景配置、确定性仿真、三维场景、遥测曲线和回放控制。
- 使用 Web Worker 执行单场景和算法对照实验，避免阻塞界面。
- 支持场景预设、参数编辑、算法切换、seed 修改及 JSON 结果导出。
- 增加模型版本、配置指纹、运行标识和公开机理边界说明。
- 修复 `SECURED` 状态与物理稳定判据不同步导致的提前终止问题。
- 增加终态一致性回归测试，并完善接管审计、架构与验证文档。

### 验证

- TypeScript 类型检查通过。
- 7 个测试文件、47 项测试通过。
- 生产构建通过。

## v0.3.0 — 故障注入、可验证实验与参数追溯（2026-07-14）

### 中文版本描述

本版本把代理仿真实验台从“可运行闭环”推进为“可注入故障、可检查数值可信度、可迁移场景、可追溯参数证据”的实验环境。所有新增结论仍限定在公开机理代理模型内部，不外推到真实型号。

### 主要修改

- 新增定时故障计划：无线静默、单绞盘卡滞、箭上导航偏置阶跃和推力降额。
- 新增无线/现场总线重复包与 CRC 损坏率界面，故障启停写入事件日志。
- 新增批量算法对照逐运行进度显示与可取消任务。
- 新增步长收敛验证：同一场景使用 Δt、Δt/2、Δt/4 比较终态与关键连续指标。
- 新增接触阶段代理能量账本，拆分接触净做功、末态弹性能和阻尼耗散代理，显式报告未观测残差，不将其包装成结构能量认证。
- 新增场景 JSON 导入、结构校验和 v1 → v2 迁移；拒绝未来未知 schema。
- 新增“参数—证据—敏感性”追溯页，支持十项关键参数的确定性单因素局部敏感性系数计算。
- 新增无界面 `lab:validate` 与 `lab:sensitivity` 入口，便于保存 JSON 实验记录。
- 场景 schema 升级到 v2，模型版本与三个工作区包版本统一为 0.3.0。

### 兼容性

- 可导入 v1 原始场景配置。
- 可导入 v0.2 导出的 `recovery-proxy-scenario` 文档包。
- 导出格式升级为 documentVersion 2。

- TypeScript 类型检查通过。
- 9 个测试文件、59 项测试通过。
- 生产构建通过。
- 标称单场景、数值验证和参数敏感性命令行烟雾测试通过。
