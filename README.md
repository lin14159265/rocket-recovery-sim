# 长征十号乙海上网系回收协同控制实验台

这是一个本地离线的候选工程模拟器：火箭、回收协调器和四个网绳/绞盘节点通过带延迟与丢包的代理链路交换消息，控制器依据带噪声估计值完成交会预测、网中心跟踪、半间距收拢和接触后的张力控制。

**在线演示：** [打开 GitHub Pages 实验台](https://lin14159265.github.io/rocket-recovery-sim/)

**当前网页版本：** v0.5.0（仿真模型仍为 v0.4.0）

> 重要：本项目仅依据公开报道和用户提供的演示截图建立代理模型。真实型号的通信协议、传感器、控制律、终端质量/速度、索网参数和载荷没有公开；本项目不代表中国航天科技集团或长征十号乙的真实实现。

## 已确认的公开机理

- 2026 年 7 月 10 日，长征十号乙完成一子级海上网系回收。
- 公开机制是回收船上的“井”字形高强度缓冲阻拦索配合箭上挂索机构捕获箭体。
- 官方报道确认了高精度导航控制、海上平台网系捕获和风浪扰动下的高动态对接，但没有公开其具体算法与参数。

来源与参数分级见 [公开资料与模型边界](docs/公开资料与模型边界.md)。
v0.5 的概念图差异、兼容性取舍与三档视口检查见 [视觉 QA 记录](docs/v0.5视觉QA.md)。

## 当前能力

- 三维刚体终端下降、风浪扰动、平台运动和四绳等效接触。
- 火箭—平台无线代理链路与平台—绞盘现场总线代理。
- 带时间戳/有效期/CRC 的消息、时延、抖动、丢包、乱序和重复。
- 固定网、α–β、预测 PD 与约束 MPC 四种候选算法，可在同一扰动与 seed 下做闭环对照。
- 制导感知滚动预测、版本绑定 READY/COMMIT 握手、四绞盘局部 PI 张力环和 MPC 异常回退。
- 算法工作台显示交会误差与 3σ 可达域、网中心指令/实际响应、四绳目标/实际张力、约束激活和求解诊断。
- 播放、暂停、前后单帧、拖动回放、实时曲线和事件日志。
- 参数来源分级、模型边界说明、相同 seed 重算和版本化 JSON 导入/导出。
- 四类定时故障注入：无线静默、单绞盘卡滞、导航偏置阶跃和推力降额。
- Web Worker 后台单场景、有界扰动算法对照、步长/能量验证和局部敏感性计算。
- 批量实验逐运行进度、取消操作与结果汇总。
- 隔离随机项、统一结束时刻的 Δt、Δt/2、Δt/4 步长收敛检查，以及物理 tick 接触能量账本。
- 双侧“参数—证据—敏感性”追溯表、终态模式边界提示与 Node 无界面复现实验入口。
- 本地离线的风暴后暖光天空、Gerstner 海浪、湿甲板/钢桁架、分段箭体、连续四索受力变形和电影跟随镜头。
- 三维显示画质支持自动/高/中/低档及 30 秒迟滞调节；画质偏好不写入场景 JSON，不参与仿真结果指纹。
