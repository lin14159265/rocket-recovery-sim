# v0.5 视觉 QA 记录

本记录只评估表现层，不构成真实型号结构或物理验证。对照基准为 v0.5 主屏概念图与海上材质参考图；仿真快照、控制算法和结果指纹未因视觉调整改变。

## 概念图差异与修复

| 对照项 | 初始差异 | 本轮修复 | 保留边界 |
|---|---|---|---|
| 主镜头构图 | 火箭过小，捕获框架遮挡主体 | 跟随镜头按高度连续改变距离、焦点和 FOV，终端阶段保持箭体、捕获平面和开放回收井同时可见 | 不使用突兀切镜或全屏运动模糊 |
| 箭上挂索点 | 视觉圆环接近箭体中部 | 挂索加强环移到鼻端以下约箭长五分之一，并加入加强带、紧固点和独立材质 | 仍是视觉代理，接触力仍作用于质心等效模型 |
| 火箭表面 | 单一光滑圆柱缺少尺度感 | 增加分段接缝、纵向面板缝、发动机簇、喷口陶瓷色、烟熏和盐渍遮罩 | 未使用真实型号 CAD 或官方材料参数 |
| 平台结构 | 简单方框缺少海工设备语义 | 增加开放回收井、桁架斜撑、湿甲板、四组绞盘、卷筒、导向轮、护栏和甲板警示标识 | 几何为低多边形工程代理，不声称复原“领航者”号结构 |
| 海况与光照 | 纯色背景、无远海层次 | 使用本地暖光风暴云层、确定性 Gerstner 波、海雾、边缘浪花和 ACES Filmic 色调映射 | 为保证 Windows ANGLE 稳定，未启用会产生黑白帧的 GTAO/Bloom 合成链 |
| 阻拦索结构 | 霓虹线条且接触变形弱 | 保持两根 X 向、两根 Y 向的连续井字索；接触后按载荷弯曲、振动并由钢缆灰过渡到警告琥珀 | 缆索形变不反向修改物理对象 |
| 控制台统一 | 三维状态层遮挡较重，色彩语义混杂 | 改为低遮挡玻璃层；青色表示控制量、绿色表示成功、琥珀表示载荷；显示实际画质档 | 信息架构和图表顺序不重排 |

## 浏览器检查

- `1368×1120`：无页面横向溢出，终端火箭与上部挂索环清晰可辨。
