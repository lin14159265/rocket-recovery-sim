# 验证计划

## 数值基础

- 自由落体、恒力加速度和悬停与解析解对照。
- 四元数持续归一，零力矩时角动量不出现非物理发散。
- 等效绳静态伸长满足 `x = mg/k`，接触力方向满足作用力与反作用力。
- 基础步长减半后：捕获时刻、末端速度变化 < 1%，峰值力变化 < 5%。

## 通信与确定性

- 同一配置与 seed 连跑两次，事件序列和量化轨迹一致。
- 验证固定时延、抖动边界、丢包、重复、乱序、CRC 错误和过期拒绝。
- 陈旧/重复命令拒绝率 100%，黑障后按状态机降级或中止。

## 闭环对照

- 固定网 / 无协同。
- α–β + PD 基线。
- Kalman + 交点预测 + 自适应收网。

相同扰动下，闭环必须显著降低捕获平面的横向误差；关闭链路或卡住绞盘后不能仍然“神奇成功”。

## 场景矩阵

- 名义捕获、收网过晚、阻尼不足、强度超限。
- 无线 20/100/250 ms 延迟，0/5/20% 丢包，1 s 黑障。
- 风浪增强、初始横向偏差/速度、质量和气动误差。
- 单绞盘迟滞/卡死、传感器偏置和量测中断。
- 200–1000 个 seed 的 Monte Carlo。

## 指标

- 捕获成功率、捕获平面脱靶量、相对速度、倾角。
- 峰值总力、单绳峰值张力、表观载荷、制动距离、稳定时间。
- 链路 p50/p95/p99 时延、有效丢包、迟包拒绝和最长黑障。
- 控制计算耗时与实时因子。

第一阶段软件门槛：名义场景必过；名义 Monte Carlo 捕获率 >= 95%；定义为不可行的场景必须在接触前中止或明确失败。以上是软件开发门槛，不是实际型号指标。
